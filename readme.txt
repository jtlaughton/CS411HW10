# Instructions

This project is built with python 3 so make sure you have that installed. Put the csv file you wish to train it with in the folder where the script is.
Make sure it follows the example of "restaurants.csv" and is for restaurants. Open a terminal and navigate to the folder where the python script is. 
Type in "python learnDecisionTree.py" to run it. It will prompt you for a filename enter the name of the file you wish to train with along with the file extension.
So if your csv file is named example type example.csv.
